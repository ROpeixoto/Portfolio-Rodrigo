Projetos

Ticket Generator - Frontend Mentor

Desafio da plataforma online Frontend Mentor, onde o objetivo consistia em recriar um site seguindo o design proposto pela plataforma. Neste desafio, o usuário deveria inserir uma imagem, e algumas informações, como o seu nome completo, email e conta do Github, e como resultado, o site encaminharia o usuário para outra janela, com um ticket gerado para o evento. O projeto foi desenvolvido com HTML, CSS e JavaScript. E está disponível para acesso no meu github e na pré visualização da tela ao lado.

Calculadora de Idade - Frontend Mentor

Desafio da plataforma online Frontend Mentor, no qual o objetivo consistia em recriar um site seguindo o design proposto pela plataforma. Neste desafio, o usuário deveria criar uma calculadora de idade, onde seria inserida a data de nascimento do indivíduo, e em retorno, iria aparecer a idade do usuário em anos, meses e dias. O projeto foi desenvolvido com Vite, CSS, Node.js e JSX. E está disponível para acesso no meu github e na pré visualização da tela ao lado.

Site com integração da API IMDB

Desenvolvimento de um site em equipe (ainda em desenvolvimento), utilizando da API entitulada IMDB a fim de fazer buscas em certos filmes e retornar a trilha sonora dos mesmos. O site está disponível no meu github e na pré visualização da tela ao lado.

Iniciação Científica Voluntária (ICV)

Projeto de iniciação científica na área de Deep Learning e IOT. Realizo uma pesquisa que envolve a revisão bibliográfica do estado da arte de redes neurais na àrea de trafego urbano, e também a criação de um sistema inteligente de um sistema que irá contribuir com a redução de congestionamentos, e, consequentemente na ampliação da mobilidade em áreas urbanas. Neste projeto foi utilizado a ferramenta SUMO, aliada a API TraCI e a linguagem de programação Python.

Jogo UNO em C

Criação de um programa na linguagem de programação C durante a matéria de Estrutura de dados. O trabalho consiste em uma simulação do jogo de cartas UNO, utilizando estruturas de dados como Árvore binária e Pilha, além do uso de outras bibliotecas e ponteiros.
